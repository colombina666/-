# Отчет по выплатам из Excel

Локальное веб-приложение на Streamlit для формирования текстового отчёта по закрытым заявкам на выплаты.

## Что Нужно В Excel

В файле должны быть столбцы:

- `страна`
- `канал выплаты`
- `сумма`
- `поставщик`

Названия столбцов можно писать в разном регистре, например `Поставщик` или `поставщик`. Лишние пробелы в названиях и значениях будут убраны автоматически.

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

После запуска откройте адрес, который Streamlit покажет в терминале. Обычно это `http://localhost:8501`.

## Как Пользоваться

1. Загрузите Excel-файл.
2. Выберите дату отчёта.
3. При необходимости измените интервал в заголовке.
4. Нажмите `Сформировать отчёт`.
5. Скопируйте текст из поля или скачайте отчёт в формате `.txt`.

## Формат Отчёта

```text
Отчет по закрытым заявкам на выплаты за 20.06.2026 с 9 до 15:00

Россия (СБП): BestPay - 125000 \ 8 шт
Казахстан (Kaspi): FastPay - 98000 \ 5 шт
```
